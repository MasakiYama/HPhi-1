We have to add new variable (new_val in this case) as

else if (strcmp(keyword, "new_val") == 0) StoreWithCheckDup_i(keyword, value, &StdI->new_val);

where StoreWithCheckDup_i() is for the integer variable; for other type, please refer the above link.

If it should be shared

If the inputted variable should be shared among routines in Standard mode, we have to add it to the list in StdFace_vals.h.

Also, the variable should be intialized before it is read. This initiallization is performed in the function StdFace_ResetVals(). We have to initialize new variable in this function as:

StdI->new_val = NaN_d;

for the float,

StdI->new_val = NaN_i;

for the integer, and

strcpy(StdI->new_val, "****\0");

for the string.
